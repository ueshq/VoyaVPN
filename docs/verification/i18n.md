## Import Process

Run:

```sh
pnpm i18n:import
```

The importer reads the reference ResX files, preserves the existing app-shell locale JSON outside the generated `resx` namespace, and rewrites `src/i18n/locales/*.json` deterministically.

Drift check:

```sh
pnpm i18n:check
```

Set `VOYAVPN_V2RAYN_RESX_DIR=/path/to/ServiceLib/Resx` to import from a non-sibling reference checkout.

## RTL Evidence

RTL is toggled through `changeLocale("fa")`, which sets:

- `document.documentElement.lang = "fa"`
- `document.documentElement.dir = "rtl"`
- `<main dir="rtl">` through `AppShell`

Switching back to an LTR locale such as `de` resets `dir` to `ltr`.
